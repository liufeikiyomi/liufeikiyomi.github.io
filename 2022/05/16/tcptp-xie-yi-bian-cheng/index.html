TCP/TP协议编程

蒋敏网课篇

面向连接的套接字——TCP套接字

socket=套接字

分为

  • TCP——面向连接的
  • UDP——面向无连接的

LInux下套接字编程

1、创建套接字

调用socket函数创建套接字（安装电话）

#include <sys/socket.h>
int socket(int domain, int type, int protocol);
//创建成功则返回文件描述符，失败返回-1。

2、分配地址信息

调用bind函数分配IP地址和端口号（分配电话号码）

#include <sys/socket.h>
int bind(int sockfd, struct sockaddr *myaddr, socklen_t addr);
//成功返回0，失败返回-1。

3、可接受连接状态

调用listen函数转为可接受请求状态。（连接电话线）

#include <sys/socket.h>
int listen(int sockfd, int backlog);
//成功返回0，失败返回-1.

4、接受请求

调用accept函数受理连接请求。（拿起话筒）

#include<sys/socket.h>
int accept(int sockfd, struct sockaddr *addr, socklen_t *addrlen);
//成功返回文件描述符，失败返回-1.

Windows网络编程

同时学Linux和Windows的原因

  • 大多数项目都在Linux操作系统下开发服务器端，而大多数客户端是在Windows平台下开发的。

  • 有时候应用程序需要在两个平台之间相互切换。

  • 两个平台下的套接字编程非常类似，学习效率比较高。

准备工作

winsock初始化

  • include头文件 winsock2.h
  • 链接ws2_32.lib库#pragma comment(lib,"ws2_32.lib"
  • Winsock初始化
#include <winsock2.h>
//设置winsock版本信息
//成功时返回0，失败时返回非零的错误代码值
int WSAStartup(WORD wVersionRequested, LPWSADATA lpWSAData);

第二个参数lpWSADATA，需传入WSADATA型结构体变量地址（LPWSADATA是WSADATA的指针类型）。调用完函数后，相应参数中将填充已初始化的库信息

int main(int argc, char* argv[]){

		ErrorHandling("WSAStartup() error!");

	if (hServSock == INVALID_SOCKET)
		ErrorHandling("socket() error");

	servAddr.sin_family = AF_INET;
	servAddr.sin_addr.s_addr = htonl(INADDR_ANY);
	servAddr.sin_port = htons(atoi(argv[1]));

	//bind分配地址
	if (bind(hServSock, (SOCKADDR*)&servAddr, sizeof(servAddr)) == SOCKET_ERROR)
